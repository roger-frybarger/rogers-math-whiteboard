Settings:

Note: Since this app is designed to be portable, these settings will only remain in effect until you close the app. Unfortunately, that means you will need to re-adjust them to your liking every time you start the app. Portability comes with its costs.

  • Remove mouse pointer over drawing area
  • Undo History: Min: 10, Max: 100
  • Max Pages Allowed: Min: 200, Max: 999
  • Enable keyboard shortcuts. (Some of these are case-sensitive)
    (ctrl+z --> Undo,
    ctrl+y --> Redo,
    ctrl+c --> Copy section,
    ctrl+v --> Paste,
    ctrl+s --> save,
    ctrl+a --> Select all,
    ctrl+Shift+a --> Deselect all,
    p --> Pen,
    e --> Eraser,
    l --> Straight Line,
    s --> Select,
    i --> Identifier,
    d --> Dot,
    b --> Blue,
    k --> Black,
    r --> Red,
    g --> Green,
    w --> White,
    Space --> Next Page,
    Shift+Space --> Previous Page,
    esc --> Cancel draw/select/paste/etc,
    Delete --> Erase selection.)
  • Optimize for wide-screen use where possible.
  • Silence error messages.
    (This can be useful if you are constantly being bothered by an error message while the program seems to be performing fine. However, it could also lead to further malfunctioning, which could leave the app unusable. Use with caution.)
  • More settings coming soon!

Here is the error log for this program. In the event that an unexpected error occurs, debugging information will be put into the box below, and you will likely receive an error message. While we don’t expect you to understand this information, it is extremely useful for diagnosing and fixing such errors. Thus, if you should receive an error message in the course of using this program please, (if possible), do the following:
1. Check rogersmathwhiteboard.com to see if you are using the latest stable version of this program. (the version number of this copy should be shown at the top of the below textbox).
2. If you are not using the latest stable version, download and try out the latest stable version to see if the problem still exists. If the problem still occurs, please report the issue by:
a. Copying the text from the below text box
b. Emailing the copied text, along with a description of the problem, to rogersmathwhiteboard@gmail.com. Doing so will greatly help us in our quest to resolve the issue, and we promise that you will never be spammed for emailing us.
Alternatively you can simply silence these error messages using the checkbox above. However, as it indicates, you should use this with caution.

Open Images

If you have not already done so, please read through the following notes about how this program opens images:

  1. It is possible to import PowerPoint content into this program. The basic steps to accomplish this are:
    1. Create a temporary folder somewhere on your computer.
    2. Open the file that you wish to import using PowerPoint or LibreOffice/OpenOffice Impress.
    3. If you are using PowerPoint, save the file into the temporary folder using the PNG format and specify that you wish to export all the slides if it asks you.
      If you are using LibreOffice/OpenOffice Impress, Export the document into the temporary folder using the HTML format, and specifying PNG images with the largest resolution available. (The HTML Design name does not matter.)
    4. Open Roger's Math Whiteboard, (This program), and import all the images in the temporary folder, or a sub-folder if applicable, using this window.
    5. Once you have successfully imported all the images, the temporary folder, and its contents, can be deleted.
    6. Need more detail? Here are two videos showing this process using PowerPoint and LibreOffice/OpenOffice Impress
  3. Also remember that Image files can be quite large. Thus; depending on the size and quantity of files you try to open, and the speed of your computer, this process may take several minutes.
  4. Since each image that you select will be opened as a page, you will need to be aware of the order in which they are selected. If they are not in the correct order when you import them, the pages will probably be out of order. Look for "Order by" or "Sort by" on the "Open Files" window to correct this sorting order. Also note that you can import individual images if needed. To do this use the "Insert Page" menu, choose "Other", and then choose "From Image."
  5. Remember that only valid PNG files smaller than 25MB in size will be imported into this program. All other files, including images that are not in PNG format, will be ignored even if you select them.
  6. For reference, the keyboard shortcut to select all files is: ctrl+a This means while holding down the ctrl key, press the letter "A." Once you see all the files selected, you can release both keys. If you are using a Mac, you may need to select the first file in the list with your mouse and then, while holding down the Shift key, use the down arrow key to select all of the files. Once all of the files are selected, you can release both keys.

Save Images

If you have not already done so, please read through the following notes about how this program saves images:

  1. Warning: if there are any existing files in the folder that you select, they may be deleted. It is best to select an empty folder or create a new empty folder to put the images in.
  2. Each page in the current document will be saved as an individual PNG image file in the folder that you choose.
  3. The text that you specify below will get added to all of the PNG Image file names. This is useful if, for example, you want all of the images to have names such as "Page1.png, Page2.png, Page3.png..." etc. However, only uppercase & lowercase letters, numbers, dashes, and underscores are accepted in this text box, so if you wish to add a space, you will need to use an underscore or dash instead.
  4. If you wish to give your students a single file instead of many individual image files, (which is highly recommend), you can add all of the image files to a zip file by selecting all of them, (The keyboard shortcut to do this is Ctrl+a), right-clicking over one of them, and choosing to send them to a compressed (zipped) folder. Your students can then download this single zip file, and extract the contents into a folder on their computer. Here are some links that explain this process in detail on Windows & Mac, Chromebooks, and Ubuntu.
  5. Also, if you plan to distribute these images using zip files, please see the detailed instructions for viewing & printing images in zip files in the "Help" section of the "File" menu. These instructions are designed to be distributed to students, so that they can make full use of the content.
