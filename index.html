Settings:

Note: Since this app is designed to be portable, these settings will only remain in effect until you close the app. Unfortunately, that means you will need to re-adjust them to your liking every time you start the app. Portability comes with its costs.

  • Remove mouse pointer over drawing area
  • Undo History: Min: 10, Max: 100
  • Max Pages Allowed: Min: 200, Max: 999
  • Enable keyboard shortcuts.
    (ctrl+z --> Undo,
    ctrl+y --> Redo,
    esc --> Cancel draw/select/paste/etc,
    alt+p --> Pen,
    alt+e --> Eraser,
    alt+l --> Straight Line,
    alt+s --> Select,
    alt+i --> Identifier,
    alt+d --> Dot,
    alt+b --> Blue,
    alt+k --> Black,
    alt+r --> Red,
    alt+g --> Green,
    ctrl+a --> Select all,
    ctrl+Shift+a --> Deselect all,
    ctrl+c --> Copy section,
    ctrl+v --> Paste,
    Delete --> Erase selection.)
  • Silence error messages.
    (This can be useful if you are constantly being bothered by an error message while the program seems to be performing fine. However, it could also lead to further malfunctioning, which could leave the app unusable. Use with caution.)
  • More settings coming soon!
